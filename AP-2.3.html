AP 2.3 - M2L Création d’un site web statique

Description du projet

Création d’un site web statique

(HTML / CSS)

Contexte : La maison des ligues de Lorraine (M2L) souhaite disposer d’un site statique (simples pages HTML/CSS, sans connexion à une base de données) pour présenter ses activités, ses ressources et ses ligues. Ce site vitrine permettra ainsi d’augmenter la visibilité des ligues et de permettre à plus de personnes de la rejoindre.

Pour cet AP, nous devions par groupe, développer un site vitrine.

Résultats attendus : Nous devions réaliser le site web statique.

Analyser les objectifs et les modalités d’organisation d’un projet : Nous commencions par nous répartir les différentes tâches et missions avant de travailler. Pour savoir quelle mission attribuer à un membre du groupe, nous analysions les objectifs demandés afin de savoir si nous étions en capacité à faire cette partie.

Planifier les activités : Une fois les différentes tâches répartis aux membres de l'équipe, nous estimions notre temps de travail pour la séance en utilisant trello, un outil de gestion de projet en ligne. Cela nous permettait d'avoir une vue d'ensemble de notre projet ainsi que son avancement chaque semaine. Avec cet outils, nous étions sûr de rendre le projet totalement terminé à la date de livraison demandée.

Traiter des demandes concernant les applications : Pour cet AP, nous devions développer un site vitrine suite aux demandes du client.
